How to run: start the live server with index.html

All of the html files are associated with the following pages:

    Nav bar:
        - index --> home page
        - profile --> your profile (still very work in progress) 
        - games --> notable games (still very work in progress)
        - log --> actual page where reviews are written (nonfunctional for now)
        - login --> takes you to login.html
            - login --> allows you to enter credentials to return to profile.html, or navigate to createAccount
            - createAccount --> allows you to enter credentials to be saved and then returns to login
        
    other: 
        - gamedetails --> (nonfunctional, however if you start the server on it you can see it) ideally allows you to see any information needed   on a game you click on. has placeholder info for the time being
        - edit-profile --> navigated to from profile.html, and simply displays 5 profile pictures (all found via the free image website) to pick from to be your profile picture. once a picture is clicked you are taken back to profile.html.
